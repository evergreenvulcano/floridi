# floridi
Agentic content wrangler environment for windows hypervisor. Pluralistic generative playground tailored for customization and adaptability. Archetypal interchangible hierarchies grounds liquid reason in edifying participatory contact surfaces. 

“interchangeable hierarchies”
could become
pluggable agent governance modules

“liquid reasoning surfaces”
could become
dynamic shared memory spaces or semantic workspaces

“participatory contact surfaces”
could become
UI panels or agent‑to‑agent negotiation protocols

etc.

Host (trusted)
 └─ Hyper‑V VM (sandbox)
     ├─ Agent A (process)
     ├─ Agent B (process)
     ├─ Agent C (process)
     └─ Shared workspace


a controlled virtual lab
with multiple agents
safe persistence
human‑approved extraction
and zero catastrophic failure modes

Agents never touch your real files.
They work in a virtual copy, and only approved results leave the sandbox.
